# 🎵 Music Player

A browser-based music player with a custom playlist and album art display, built using vanilla HTML, CSS, and JavaScript.

---

## ✨ Features

- ▶️ Play, pause, and skip through a curated playlist
- 🎨 Dynamic album art display synced to the currently playing track
- 🎚️ Custom-built player UI and controls — no external player libraries
- 📱 Responsive, clean interface

---

## 🛠️ Tech Stack

`HTML5` · `CSS3` · `JavaScript (Vanilla)`

Built with the native HTML5 `<audio>` element and vanilla JS for playback control, playlist logic, and UI state — no frameworks or external player libraries.

---

## 🚀 Getting Started

```bash
git clone https://github.com/mabdullahabrar/Music-Player.git
cd Music-Player
```

Open `index.html` in your browser to run the player — no build step or server required.

---

## 📁 Project Structure

```
Music-Player/
├── index.html      # App structure & markup
├── script.js       # Playback logic, playlist control, UI updates
├── style.css        # Styling
└── *.jpeg           # Album art assets for each track
```
